## 안녕하세요! 👋

**주경야근** 블로그에 오신 것을 환영합니다.

이 공간은 프로그래밍, 데이터분석, IT 트렌드에 관한 기술 블로그입니다. 낮에도 밤에도 코딩하는 개발자의 경험과 지식을 공유합니다.

## 블로그 소개

이 블로그는 다음 기술들로 만들어졌습니다:

- **Eleventy** - 빠르고 유연한 정적 사이트 생성기
- **Tailwind CSS** - 모던한 디자인을 위한 CSS 프레임워크
- **Nunjucks** - 강력한 템플릿 엔진
- **Prism.js** - 코드 하이라이팅
- **Netlify** - 자동 배포 및 호스팅

## 주요 카테고리

- **Frontend** - React, Vue, JavaScript, TypeScript 등
- **Backend** - Node.js, Python, API 개발 등
- **Modeling** - 데이터 모델링, 머신러닝 등
- **NoSQL** - MongoDB, Redis 등 NoSQL 데이터베이스
- **DevOps** - CI/CD, Docker, Kubernetes 등
- **AI/ML** - 인공지능, 머신러닝, 딥러닝 등

## 특징

- 📝 **코드 중심**: 실용적인 코드 예제와 스니펫
- 📊 **데이터 시각화**: 차트와 다이어그램을 활용한 설명
- 🔍 **검색 기능**: 원하는 내용을 빠르게 찾기
- 🏷️ **태그 시스템**: 관련 글을 쉽게 탐색
- 💬 **댓글 기능**: GitHub 기반 댓글로 소통
