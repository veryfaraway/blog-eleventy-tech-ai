# 블로그 수익화 가이드

## Google Analytics & AdSense 설정

### 1단계: Google Analytics 설정

#### 계정 생성
1. [Google Analytics](https://analytics.google.com/) 접속
2. "측정 시작" 클릭
3. 계정 이름 입력
4. 속성 설정 (웹사이트 이름, URL, 시간대)
5. 데이터 스트림 생성 (웹)
6. **측정 ID 복사** (G-XXXXXXXXXX)

#### 블로그에 적용
`src/_data/site.json` 파일 수정:
```json
{
  "googleAnalytics": "G-XXXXXXXXXX"
}
```

#### 확인
- 사이트 배포 후 방문
- Analytics 실시간 보고서에서 확인

---

### 2단계: Google AdSense 설정

#### 계정 생성 및 승인
1. [Google AdSense](https://www.google.com/adsense/) 접속
2. 계정 생성 및 사이트 등록
3. 결제 정보 입력
4. **승인 대기** (1-2주 소요)

#### 승인 요건
- ✅ 고품질 콘텐츠 (최소 20-30개 포스트 권장)
- ✅ 정기적인 업데이트
- ✅ 독창적인 콘텐츠
- ✅ 명확한 네비게이션
- ✅ 개인정보 보호정책 페이지
- ❌ 저작권 침해 콘텐츠 없음
- ❌ 성인 콘텐츠 없음

#### 광고 단위 생성
승인 후 AdSense 대시보드에서:

1. **광고 > 광고 단위** 메뉴
2. **디스플레이 광고** 생성
   - 이름: "Display Ad"
   - 광고 크기: 반응형
   - **data-ad-slot 번호 복사**
3. **인피드 광고** 생성
   - 이름: "In-Article Ad"
   - 광고 크기: 반응형
   - **data-ad-slot 번호 복사**

#### 블로그에 적용
`src/_data/site.json` 파일 수정:
```json
{
  "googleAnalytics": "G-XXXXXXXXXX",
  "adsense": {
    "enabled": true,
    "client": "ca-pub-1234567890123456",
    "slots": {
      "inArticle": "1234567890",
      "display": "0987654321"
    }
  }
}
```

**필드 설명:**
- `enabled`: `true`로 설정하여 광고 활성화
- `client`: AdSense 게시자 ID (AdSense 대시보드에서 확인)
- `slots.inArticle`: 인피드 광고 슬롯 ID
- `slots.display`: 디스플레이 광고 슬롯 ID

## 광고 삽입 방법

### 자동 광고 (권장)
`site.json`에서 `enabled: true`로 설정하면 Google이 자동으로 최적 위치에 광고 배치

### 수동 광고 삽입

#### 인피드 광고 (글 중간)
포스트 마크다운 파일에서:
```markdown
## 섹션 1

충분한 내용 작성 (최소 300-500자)...

{% adsense "inArticle" %}

## 섹션 2

계속 내용 작성...
```

#### 디스플레이 광고 (일반)
```markdown
{% adsense "display" %}
```

---

## 광고 배치 전략

### 긴 포스트 (2000자 이상)
```markdown
# 제목

서론 (300-500자)

## 첫 번째 섹션
내용 (500-800자)

{% adsense "inArticle" %}

## 두 번째 섹션
내용 (500-800자)

{% adsense "display" %}

## 결론
마무리
```

### 중간 포스트 (1000-2000자)
```markdown
# 제목

서론 및 본문 (800-1500자)

{% adsense "inArticle" %}

## 결론
마무리
```

### 짧은 포스트 (1000자 미만)
- 광고 사용 자제 권장
- 사용 시 1개만 포스트 끝에 배치

---

## 광고 최적화 팁

### 1. 콘텐츠 품질
- 고품질 콘텐츠가 더 높은 CPC 유도
- 정기적인 업데이트
- 독창적이고 유용한 정보 제공

### 2. 광고 배치
- 자연스러운 위치에 배치
- 첫 화면에 광고만 가득하지 않도록
- 콘텐츠와 광고 비율: 80:20 권장

### 3. 사용자 경험
- 페이지 로딩 속도 최적화
- 모바일 친화적 디자인
- 광고로 인한 레이아웃 깨짐 방지

### 4. 트래픽 증가
- SEO 최적화
- 소셜 미디어 공유
- 이메일 뉴스레터
- 게스트 포스팅

## AdSense 정책 준수

### 금지 사항
- ❌ "광고를 클릭하세요" 같은 유도
- ❌ 광고 주변에 화살표나 강조
- ❌ 자신의 광고 클릭
- ❌ 클릭 교환 프로그램 참여
- ❌ 부적절한 콘텐츠

### 권장 사항
- ✅ 자연스러운 광고 배치
- ✅ 고품질 콘텐츠
- ✅ 사용자 경험 우선
- ✅ 정책 정기 확인

## 성과 측정

### 주요 지표

#### 1. 페이지 RPM (Revenue Per Mille)
- 1000회 노출당 수익
- 목표: $5-$20 (주제에 따라 다름)

#### 2. CTR (Click Through Rate)
- 클릭률
- 평균: 1-3%
- 높을수록 좋음

#### 3. CPC (Cost Per Click)
- 클릭당 비용
- 주제에 따라 $0.1-$5+
- 금융, 보험 등이 높음

#### 4. 노출수
- 광고 표시 횟수
- 트래픽과 비례

### 수익 예상

**예시 계산:**
- 일일 방문자: 1,000명
- 페이지뷰: 2,000 (방문자당 2페이지)
- CTR: 2%
- CPC: $0.50

**일일 수익:**
```
클릭 수 = 2,000 × 0.02 = 40 클릭
수익 = 40 × $0.50 = $20
```

**월간 수익:**
```
$20 × 30일 = $600
```

## 문제 해결

### 광고가 표시되지 않음
1. `site.json`에서 `enabled: true` 확인
2. AdSense 계정 승인 확인
3. 광고 코드 정확성 확인
4. 브라우저 광고 차단기 비활성화
5. 24-48시간 대기 (새 사이트의 경우)

### 수익이 낮음
1. 트래픽 증가 노력
2. 고가치 키워드 타겟팅
3. 광고 배치 최적화
4. 콘텐츠 품질 개선
5. 페이지 로딩 속도 개선

### 정책 위반 경고
1. 즉시 문제 수정
2. AdSense 팀에 재검토 요청
3. 정책 숙지 및 준수
